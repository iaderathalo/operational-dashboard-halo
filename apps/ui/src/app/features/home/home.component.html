Getting Started

This page is a Polaris design system built using Marsh brand guidelines to help you quick start your project. Below is a series of instructions and guides to help you get started with the Polaris MEAN Blueprint and use it as the starting point for your next application.

UI


The UI is a responsive Angular application based largely Nx - an extensible framework used for assisting with template code generation and application build procedures.

For an overview of the structure of the UI, including any design patterns and development approaches to follow and for guidance on how to extend the Blueprint, including adding new components and pages, see the Polaris MEAN Blueprint Structure documentation

MICRO LenAI


You can find Micro LenAI in the bottom right corner of the page. Micro LenAI is a web component that provides a chat interface for users to interact with the application.


You need to upload your documents for Micro LenAI to answer the questions. You can see this page how to upload documents:  Micro LenAI Documentation

CRUD APPLICATION


This application contains pre-configured examples which showcase the different capabilities of the Blueprint.


Task List showcases CRUD functionality in the Blueprint though creating, editing and deleting tasks, with active tasks shown in a list format.
